剑指数据仓库早课

1、查看进程的命令，杀死进程之前应该做些什么
答：杀死进程前要确认这个进程是否能被杀，不要是别人正在运行的服务。

2、假如ps确认清楚，杀死名称含有关键词abc的所有进程，该怎么杀？
答：kill -9 $(ps -ef|grep -f abc)

3、ps -ef|grep XXX命令，可以查看进程的那些信息
答：pid

4、查看端口号的命令
答：netstat -nlp|grep

5、rpm包怎么搜索、安装、卸载
答：yum search
yum install -y
yum remove

6、tar压缩解压、参数
答：压缩：
解压：tar -xzvf

7、下载一个http文件的命令
答：wget

8、sudo和su的区别
答：su切换到家目录，sudo切换后还是在当前目录

9、一个用户无法su/ssh登录，该检查那个文件？
答：cat /etc/passwd

10、系统负载命令，一般看哪两块？
答：top

11、查看内存、磁盘的命令分别是什么？
答：free -m、df -h

12、当前目录使用什么命令进行标识？
答：./

13、绝对路径和相对路径的定义和概念？
答：绝对路径/开头，相对路径：~

14、全局环境变量文件在哪？该怎样生效？生效完成后，习惯做什么？
答：root用户下，/etc/profile；source /etc/profile，生效完成后使用which java、which hive

15、个人环境变量文件默认在哪？使用哪个文件？
答：vi ~/.bash_profile，在用户的家目录下

16、输入一个命令，抛错command not found，你们觉得应该从哪些方面检查？
答：环境变量

17、不管是作为新人还是老手，要动公司服务器上的生产配置文件，修改前要养成什么习惯？
答：先cp备份一下

18、vi编辑文件，差不多1000行，想要快速定位到某个参数port=110，去修改值port=120,；如何快速定位找到这个参数？
答：esc进入尾行模式，输入:/port=110查询然后编辑替换即可。

19、Linux的模糊匹配和MySQL的模糊匹配分别是什么标识
答：linux系统中的模糊匹配是*和？，mysql中的模糊匹配是%和_

20、隐藏文件标识是什么，该怎么查看？
答：ll -a

21、一个文件10M，想要快速找到error关键词的上下文10行，命令是什么？
答：grep ERROR -C 10 XXX

22、> 和 >>的区别是什么？
答：>表示把原来的内容全部覆盖掉；>>表示在我们原来的文件内容下进行追加

23、管道符是什么？你的理解？
答：| 表示管道符，前者的输出作为后者的输入

24、echo " > xxx.log清空文件，有什么问题？
答：实际上还占一个字节

25、简述一个流程：vi 将一个文件的内容从windows的记事本去替换掉，流程是什么？
答：vi编辑待替换文件，gg然后dG清空此文件，复制黏贴然后使用secureCRT直接右键黏贴，注意是要在编辑模式下，否则直接黏贴会丢失几个字符。
